Commercial Project
Jun 2019 - Aug 2019

For a commercial operation, I designed and coded an automated system for an industrial printshop (that produced booklets, posters, fliers, etc). I did this using the programming language Microsoft VBA (Visual Basic for Applications). This project involved replacing the slow and tedious manual entries of the printing jobs into an Excel Spreadsheet with an on-screen form containing check-boxes, radio-buttons, drop-down lists to select from, and so forth. This meant that the computer operator no longer needed to manually enter the record, a time-consuming process. Instead they merely needed several quick clicks of the mouse on the form's options. Once done, the VBA code I wrote automatically performed calculations, and then automatically generated and saved a spreadsheet record of the printing job.
